# Intro

I'm a Fullstack Software Engineer with strong experience in designing and implementing scalable applications through Java Spring Boot microservices and React-based frontends. Currently working as an Expert Software Engineer at [Yapı Kredi Technology](https://www.ykteknoloji.com.tr/).

# What I Do

On the backend, I specialize in transforming legacy Java 8 monoliths into Java 17 microservices using Spring Boot. I develop REST APIs, implement asynchronous communication with RabbitMQ, Shovel, and Kafka, and enable inter-service communication through Feign clients. I work extensively with Oracle DB, writing and maintaining PL/SQL procedures and cursors.

On the frontend, I build React applications leveraging context-based state management to create maintainable and user-friendly interfaces.

I also design and maintain Spring Boot batch jobs, manage scheduling with UC4, and take part in alerting and monitoring processes to keep business-critical operations running smoothly.

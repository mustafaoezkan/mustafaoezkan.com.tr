# Giriş

Java Spring Boot mikroservisler ve React tabanlı arayüzler ile ölçeklenebilir uygulamalar tasarlayan ve geliştiren bir Fullstack Yazılım Mühendisiyim. Şu anda [Yapı Kredi Teknoloji](https://www.ykteknoloji.com.tr/)'de Uzman Yazılım Mühendisi olarak çalışıyorum.

# Ne Yapıyorum

Backend tarafında, eski Java 8 monolitik sistemleri Java 17 Spring Boot mikroservislere dönüştürme konusunda uzmanım. REST API'ler geliştiriyorum, RabbitMQ, Shovel ve Kafka ile asenkron iletişim altyapıları kuruyorum ve Feign client'lar ile servisler arası iletişimi sağlıyorum. Oracle DB ile yoğun şekilde çalışıyor, PL/SQL prosedürleri ve cursor'ları yazıyorum.

Frontend tarafında, context tabanlı state management kullanarak bakımı kolay ve kullanıcı dostu React uygulamaları geliştiriyorum.

Ayrıca Spring Boot batch job'ları tasarlıyor, UC4 ile zamanlama yönetimi yapıyor ve iş kritik operasyonların sorunsuz çalışması için alarm ve izleme süreçlerine katılıyorum.

# Geçmiş Deneyimler

Ayrıca [Erciyes Üniversitesi](https://www.erciyes.edu.tr/) BT Departmanı'nda (React & Node.js ile envanter sistemi) ve [CTech](https://ctech.com.tr/)'te (Flutter & RTMP ile canlı yayın uygulaması) staj yaptım.
